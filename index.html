Some examples of use of basic HTML tags

HTML basic structure

The basic structure of an HTML page is:

  1. !doctype (HTML document type)
  2. HTML tag (document container)
    1. HEAD tag (document header, invisible part)
    2. BODY tag (document body, visible part)

Headings

The basic headings are:

  • H1: First level header
  • H2: Second level header
  • H3: Third level header
  • H4: Fourth level header
